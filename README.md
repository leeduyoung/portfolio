## Installation
이 프로젝트를 빌드하기 위해서는 NodeJS가 설치되어 있어야 한다.

```bash
# Install Dependencies
npm install
```

이 프로젝트를 빌드하고, 실행하려거든 아래 커맨드를 수행하면 된다.

```bash
npm run build # Compile
npm run start # Execute

# Dev Mode
npm run dev
```
